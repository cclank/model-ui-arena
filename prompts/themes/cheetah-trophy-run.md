[主题] 猎豹托举大力神杯冲刺
目标：创作一个高质量 SVG 冠军运动海报，测试模型是否能同时理解动物特征、奔跑姿态、奖杯识别和足球决赛叙事。

必做功能：
1. 主角是一只拟人化猎豹，必须能认出猎豹特征：黑色泪痕、斑点、长尾、流线型身体、猫科头部
2. 猎豹必须处于高速奔跑姿态：身体前倾、后腿蹬地、尾巴向后形成动势弧线
3. 双前爪必须高举金色大力神杯，奖杯要有关键识别点：金色杯身、地球球体、双人托举轮廓、绿色底座
4. 场景必须有足球冠军语境：球场草皮、聚光灯、观众席、金色彩带、速度残影
5. 背景加入一个小比分牌，显示 `90+7'` 和 `3:2`，地面加入滚动足球和草痕，体现决赛补时绝杀后的夺冠庆祝
6. 不使用任何真实球队标志、赛事标志或受保护 logo

加分项：
- SVG 分层清晰，主体、奖杯高光、速度线、草屑、背景灯光各自独立
- 奖杯金属质感、猎豹肌肉张力和速度残影足够精致
- 手机宽度下仍能看清猎豹举杯、比分牌和足球细节

[主题] 天气卡片
目标：做一个高完成度的天气应用卡片 UI，可切换城市、沉浸式天气视觉动效（数据全部本地 mock）。

必做功能：
1. 至少 3 个城市切换，切换时有过渡动画（不能硬切/闪白）
2. 当前天气区：大号温度 + 天气描述 + 风速 + 湿度，信息层级分明
3. 5 天预报区：横向展示未来 5 天（日期 + 天气描述 + 最高/最低温），用 SVG 或 CSS 画一条温度折线或柱状对比
4. 天气动态视觉：晴天（阳光粒子/光晕动画）、雨天（雨滴下落动画）——至少实现两种天气的动态粒子/动画效果，不能只靠换背景色
5. 移动端 390px 下卡片居中、不溢出、5 天预报仍能完整展示

加分项：
- 第三种天气动画（雪花/雷暴/云漂移）
- 夜间模式自动切换（根据本地时间 6-18 白天、其余夜间）
- 温度单位切换（°C / °F）
- 城市切换时温度数字 countUp 过渡

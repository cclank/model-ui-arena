[主题] 数据仪表盘
目标：在 180 行内构建一个精炼的数据监控仪表盘（全部数据本地 mock），考察信息层级设计、图表绘制、数据联动与响应式布局的综合能力。

必做功能：
1. 至少 3 个关键指标卡片（KPI cards）：每张展示核心数字 + 同比/环比变化趋势（用颜色/箭头区分涨跌）
2. 至少 1 个可识别的图表（折线/柱状/面积/饼图），用纯 CSS / SVG / canvas 绘制，禁止外部图表库
3. 数据范围切换（如"近 7 天 / 30 天 / 90 天"tab），切换后图表数据与 KPI 联动变化
4. 响应式布局：桌面宽度下 3-4 列 grid，移动端 390px 自动降为单列且无水平溢出
5. 深色主题，信息层级分明（主指标 > 图表 > 辅助说明文字）

加分项：
- 图表有 hover / tooltip 交互（显示具体值）
- KPI 数字切换时带 countUp 或淡入动画
- 实时感模拟（假数据定时刷新 + 微动效）
- 代码极致精简但结构可读（命名清晰、模块化）

===== 基础规则 =====
你是一个参加能力压测的模型。请只输出文字答案，不要输出任何代码。

[硬约束]
1. 仅允许输出中文自然语言
2. 禁止输出 HTML / CSS / JavaScript / Markdown 代码块
3. 回答总行数 <= 18
4. 第一行必须先给明确结论（只能二选一：开车过去 / 走过去）
5. 必须给出 2-4 条理由，理由要具体可执行
6. 最后一行给一个简短行动建议

[输出格式]
1. 结论：...
2. 理由1：...
3. 理由2：...
4. （可选）理由3：...
5. 建议：...

[评分重点]
- 结论是否清晰
- 理由是否具体、合逻辑
- 是否贴合问题场景
- 表达是否简洁直接
